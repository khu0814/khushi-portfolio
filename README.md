# Khushi Portfolio Website

A responsive single-page portfolio website hosted on AWS using Amazon S3 and CloudFront.

## Project Overview

This project was created as part of the Cloud Computing & DevOps Internship Task 1.

The objective was to build a responsive portfolio website and deploy it using:
- Amazon S3 for static website hosting
- Amazon CloudFront for CDN delivery and HTTPS security

## Features

- Responsive single-page portfolio
- Hero section with CTA button
- About section
- Projects section with 3 project cards
- Contact section
- Footer
- Google Fonts integration
- Font Awesome icons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- AWS S3
- AWS CloudFront

## Deployment Steps

1. Created an Amazon S3 bucket for static website hosting.
2. Uploaded portfolio files including HTML, CSS, JavaScript, and assets.
3. Enabled static website hosting on S3.
4. Configured bucket policy for public access.
5. Created a CloudFront distribution for HTTPS and CDN delivery.
6. Verified the website using the CloudFront URL.
